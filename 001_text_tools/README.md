# 텍스트 도구 (Text Tools)

이 폴더에는 텍스트 및 문서 처리와 관련된 유틸리티 프로그램들이 포함되어 있습니다.

## 포함된 프로그램

### SimpleNotepad
- **별도 저장소로 분리됨**: [SimpleNotepad](https://github.com/YuHyungmin1226/SimpleNotepad)
- 고급 메모장 애플리케이션
- 다크모드 지원, 자동 저장, 고급 검색 기능
- 크로스 플랫폼 지원 (Windows, macOS, Linux)
